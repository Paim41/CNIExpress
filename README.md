# Creativity & Innovation Express

An interactive and visually immersive study platform for **BIC31502 – Creativity & Innovation**, designed to make learning more engaging through structured notes, flashcards, quizzes, and progress tracking.

## 🌐 Live Demo

[Creativity & Innovation Express Live Demo](https://cni-express.vercel.app/?utm_source=chatgpt.com)

## ✨ Features

- 📘 Comprehensive chapter notes
- 🃏 Interactive flashcards with flip animations
- 🧠 Exam-style quiz engine with automatic scoring
- 📈 Progress tracking and achievement badges
- 🔍 Searchable study materials
- 🌙 Light and dark mode
- ⌨️ Keyboard shortcuts for faster navigation
- 📱 Fully responsive design
- 🎨 Futuristic sci-fi inspired interface with animated background

## 📚 Covered Chapters

1. Introduction to Creativity & Innovation
2. Creative Problem Solving (CPS)
3. Creative Problem Definition

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Orbitron, Exo 2, Share Tech Mono)

## 🎯 Learning Tools

### Notes
Well-structured chapter summaries with:
- Key definitions
- Tables and comparisons
- Summary boxes
- Mnemonics for memorization

### Flashcards
- Flip-to-reveal answers
- Shuffle mode
- Auto-play mode
- Mastered card tracking

### Quiz Engine
- Multiple-choice questions
- Timed sessions
- Instant feedback
- Detailed explanations
- Performance review

### Progress Dashboard
- Study streak counter
- Achievement badges
- Completion statistics
