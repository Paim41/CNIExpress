# CNI Express

A modern logistics and delivery service web application designed to provide efficient shipment tracking, service management, and a user-friendly experience.

Live Demo: https://cni-express.vercel.app/

---

## ▸ Overview

CNI Express is a web-based platform that simulates a real-world courier and logistics system. It allows users to explore delivery services, track shipments, and navigate through a clean and structured interface.

The project emphasizes usability, responsive design, and practical application structure.

---

## ▸ Features

- Shipment tracking interface  
- Logistics and delivery service overview  
- Clean and structured UI/UX  
- Fully responsive design  
- Fast and lightweight performance  
- Clear navigation and user flow  

---

## ▸ Use Case

This project is suitable for:

- Frontend development practice  
- Portfolio demonstration  
- Simulation of real-world logistics systems  
- UI/UX design showcase  

---

## ▸ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Styling: Custom CSS  
- Deployment: Vercel
